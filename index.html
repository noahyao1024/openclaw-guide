0前置知识 — 什么是终端？

终端（Terminal）是一个用文字和电脑对话的窗口。别被它吓到，你只需要复制粘贴命令就行。

打开终端

  • Mac：按 ⌘ + 空格，搜索"终端"或"Terminal"，回车打开
  • Windows：按 Win + R，输入 powershell，回车打开
  • Linux：按 Ctrl + Alt + T
💡 小贴士：终端里看到 $ 或 % 开头的行表示"请在这里输入命令"。复制命令时不要复制 $ 符号本身。

1安装 Node.js（运行环境）

OpenClaw 需要 Node.js 来运行。就像手机需要操作系统一样，Node.js 是 OpenClaw 的"操作系统"。

Mac / Linux

推荐用 nvm 来安装（版本管理更方便）：

# 安装 nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.3/install.sh | bash

# 重新打开终端，然后安装 Node.js
nvm install 22
node -v   # 应该显示 v22.x.x

Windows

直接去 nodejs.org 下载安装包，选择 LTS（长期支持）版本，一路下一步即可。

💡 验证安装：在终端输入 node -v，看到版本号就说明成功了。

2安装 Claude Code（AI 编程助手）

Claude Code 是 Anthropic 推出的命令行 AI 助手，可以帮你写代码、管理文件、回答问题。它也是 OpenClaw 支持的 AI 后端之一。

# 全局安装 Claude Code
npm install -g @anthropic-ai/claude-code

# 验证安装
claude --version

配置 API Key

你需要一个 Anthropic API Key 来使用 Claude：

  1. 访问 console.anthropic.com
  2. 注册账号 → 进入 API Keys 页面
  3. 点击 "Create Key"，复制生成的 key
  4. 在终端中设置：
# Mac / Linux — 添加到 shell 配置文件
echo 'export ANTHROPIC_API_KEY="你的key"' >> ~/.zshrc
source ~/.zshrc

# Windows PowerShell
$env:ANTHROPIC_API_KEY = "你的key"
⚠️ 注意：API Key 是付费的，按使用量计费。建议先充值 $5 试用。保管好你的 key，不要分享给别人。

3安装 OpenClaw（AI 工作流平台）

OpenClaw 是一个让 AI 助手"活起来"的平台——它可以连接你的消息应用、定时执行任务、管理记忆，把 AI 变成你的私人助理。

# 安装 OpenClaw
npm install -g openclaw

# 验证安装
openclaw --version

初始化配置

# 初始化（会创建 ~/.openclaw 目录）
openclaw init

# 启动 gateway 服务（后台常驻）
openclaw gateway start

# 查看状态
openclaw status
📁 目录结构说明 
~/.openclaw/
├── config.yaml      # 主配置文件
├── workspace/       # 工作空间
│   ├── SOUL.md      # AI 人格设定
│   ├── USER.md      # 你的个人信息
│   ├── MEMORY.md    # AI 长期记忆
│   └── memory/      # 每日记忆文件
└── skills/          # 技能插件

配置文件详解

打开 ~/.openclaw/config.yaml，这是最核心的配置文件：

# ~/.openclaw/config.yaml 基础配置示例
model: deepseek/deepseek-chat    # 默认模型（见第4步）

# API 提供商配置
providers:
  deepseek:
    apiKey: "你的DeepSeek API Key"
    baseUrl: "https://api.deepseek.com"

4选择 & 配置中文模型

作为中文用户，你需要一个中文能力强、价格合理、国内可访问的模型。以下是推荐排序：

模型 推荐度 中文能力 价格 特点
DeepSeek V3 ⭐ 首选 🟢 极强 极低（¥1/百万token） 中文最强、价格最低、国内直连
DeepSeek R1 ⭐ 推荐 🟢 极强 低 推理能力强，适合复杂分析
Qwen (通义千问) 备选 🟢 极强 低 阿里出品，生态完善
Doubao (豆包) 备选 🟢 强 低 字节出品，对话自然
Claude Sonnet 备选 🟡 良好 中等 综合能力强，需翻墙
🏆 强烈推荐：DeepSeek V3

对于中国用户来说，DeepSeek 是目前最佳选择：

  • ✅ 中文理解和生成能力顶级
  • ✅ 价格极低（可能是最便宜的高质量模型）
  • ✅ 国内可以直接访问，无需 VPN
  • ✅ API 兼容 OpenAI 格式，配置简单

配置 DeepSeek（推荐）

  1. 访问 platform.deepseek.com
  2. 注册账号，进入"API Keys"页面
  3. 创建一个 API Key
  4. 充值（最低 ¥10 起，可以用很久）
  5. 配置 OpenClaw：
# 方式一：环境变量（推荐）
# Mac / Linux
echo 'export DEEPSEEK_API_KEY="你的key"' >> ~/.zshrc
source ~/.zshrc

# 方式二：直接写入 config.yaml
# 编辑 ~/.openclaw/config.yaml

在 config.yaml 中配置模型：

# ~/.openclaw/config.yaml
model: deepseek/deepseek-chat

providers:
  deepseek:
    apiKey: ${DEEPSEEK_API_KEY}    # 引用环境变量
    baseUrl: https://api.deepseek.com/v1

配置 Qwen（备选）

# 通义千问配置
model: qwen/qwen-max

providers:
  qwen:
    apiKey: ${DASHSCOPE_API_KEY}
    baseUrl: https://dashscope.aliyuncs.com/compatible-mode/v1
💡 如何获取通义千问 API Key：访问 阿里云灵积平台，注册后在"API-KEY 管理"中创建。

5连接消息平台

OpenClaw 最强大的功能之一是可以通过消息应用和你的 AI 助理对话。

Telegram（推荐）

Telegram 是 OpenClaw 支持最完善的平台：

  1. 在 Telegram 中搜索 @BotFather
  2. 发送 /newbot，按提示创建机器人
  3. 获取 Bot Token
  4. 配置到 OpenClaw：
# ~/.openclaw/config.yaml
channels:
  telegram:
    token: "你的Bot Token"
    allowedUsers:
      - 你的Telegram用户ID    # 数字ID，安全起见限制访问
💡 获取你的 Telegram ID：在 Telegram 中搜索 @userinfobot，发送任意消息即可获取你的数字 ID。

微信（通过 webhook）

微信官方没有开放个人聊天 API，但你可以通过一些第三方工具（如 chatgpt-on-wechat）桥接到 OpenClaw 的 webhook 接口。这比较高级，建议先从 Telegram 开始。

?常见问题

Q: 需要翻墙吗？

如果你使用 DeepSeek 或通义千问作为模型，不需要翻墙。但 Telegram 在中国需要翻墙使用。如果无法使用 Telegram，可以考虑 webhook 方式接入其他平台。

Q: 费用大概多少？

DeepSeek API 极其便宜，日常使用每月可能只需要 ¥5-20。OpenClaw 本身是开源免费的。

Q: 没有编程基础能用吗？

完全可以！这份指南就是为非技术人员写的。你只需要在终端里复制粘贴命令。配置好之后，日常使用就是在 Telegram 里聊天。

Q: 数据安全吗？

OpenClaw 运行在你自己的电脑上，数据不经过第三方服务器（除了 AI 模型 API）。你的对话记录、记忆文件都存在本地。

Q: Mac 和 Windows 都能用吗？

都可以。Mac / Linux 体验最好。Windows 推荐使用 WSL (Windows Subsystem for Linux) 来获得更好的体验。
