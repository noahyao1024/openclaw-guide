📋 你需要准备的东西

  • 一台 Mac 或 Windows 电脑
  • 能上网（部分步骤可能需要科学上网）
  • 一个手机号（注册用）
  • 大约 30 分钟的时间

第一步 注册 DeepSeek 账号 ⏱ 5 分钟

DeepSeek 是目前中文最好的 AI 模型，国内可以直接用，而且非常便宜（充 10 块钱能用几个月）。

  1. 用浏览器打开 platform.deepseek.com
  2. 点"注册"，用手机号注册
  3. 登录后，点左侧菜单的 "API Keys"
  4. 点 "创建 API Key"，起个名字（比如"openclaw"）
  5. ⚠️ 复制生成的 Key，保存到备忘录里（它只显示一次！）
  6. 点左侧 "费用"，充值 ¥10（支付宝/微信都行）
⚠️ 重要：API Key 就像密码，不要发给别人，不要发到群里。

第二步 安装 Node.js ⏱ 5 分钟

这是 OpenClaw 运行需要的基础软件，装一次就行。

  1. 打开 nodejs.org/zh-cn
  2. 点绿色的 "下载 LTS" 按钮
  3. 双击下载的文件，一路点"继续/下一步"装完
✅ 验证：打开终端（Mac 按 ⌘+空格 搜"终端"；Windows 按 Win+R 输入 cmd），输入： 
node -v
看到类似 v22.x.x 就说明装好了。

第三步 安装 OpenClaw ⏱ 5 分钟

在刚才的终端窗口里，复制粘贴以下命令，按回车：

npm install -g openclaw

等它跑完（可能要 1-2 分钟），然后输入：

openclaw init

它会问你一些问题，按提示填就行。

第四步 配置 AI 模型 ⏱ 5 分钟

告诉 OpenClaw 用 DeepSeek 作为大脑。

Mac 用户

在终端里复制粘贴这两行（把 你的Key 换成第一步保存的 API Key）：

echo 'export DEEPSEEK_API_KEY="你的Key"' >> ~/.zshrc
source ~/.zshrc

Windows 用户

  1. 按 Win + I 打开设置
  2. 搜索"环境变量" → 点"编辑系统环境变量"
  3. 点"环境变量" → "新建"
  4. 变量名填 DEEPSEEK_API_KEY
  5. 变量值填你的 API Key
  6. 确定保存，重新打开终端

然后用文本编辑器打开配置文件 ~/.openclaw/config.yaml，加入：

model: deepseek/deepseek-chat

providers:
  deepseek:
    apiKey: ${DEEPSEEK_API_KEY}
    baseUrl: https://api.deepseek.com/v1
💡 不知道怎么打开配置文件？
Mac：在终端输入 open ~/.openclaw/config.yaml
Windows：在终端输入 notepad %USERPROFILE%\.openclaw\config.yaml

第五步 创建 Telegram 机器人 ⏱ 5 分钟

通过 Telegram 跟你的 AI 助理聊天，随时随地，手机电脑都能用。

  1. 下载 Telegram（手机或电脑都行）
  2. 在 Telegram 搜索 @BotFather，给它发消息
  3. 发送 /newbot
  4. 它会让你起个名字和用户名，随便取
  5. 复制它给你的 Token（一长串字符）
  6. 再搜索 @userinfobot，发任意消息，它会回复你的数字 ID

把这些信息加到配置文件里：

channels:
  telegram:
    token: "你的Bot Token"
    allowedUsers:
      - 你的数字ID

第六步 启动！ ⏱ 1 分钟

一切配置好了，在终端输入：

openclaw gateway start

然后打开 Telegram，找到你创建的机器人，发一条消息试试！

🎨 让 AI 更懂你（可选）

OpenClaw 有几个特别的文件，可以让 AI 记住你的信息、按你喜欢的风格说话：

👤 USER.md — 你是谁

告诉 AI 你的基本信息，它就能更好地帮你。打开 ~/.openclaw/workspace/USER.md，写入：

- 名字：王帆
- 角色：产品经理
- 公司/行业：（你的行业）
- 关注领域：AI 产品化、用户体验
- 语言：中文

🤖 SOUL.md — AI 的性格

让 AI 用你喜欢的方式说话。打开 ~/.openclaw/workspace/SOUL.md，写入：

## 核心原则
- 全程用中文交流
- 说人话，别搞技术黑话
- 像靠谱的同事一样给建议
- 简洁为主，废话少说

## 风格
- 务实，不要假大空
- 有观点，别和稀泥
- 适当幽默，但别尬

🔧 几个常用命令

不用记，用到的时候回来看就行：

# 查看运行状态
openclaw status

# 启动服务
openclaw gateway start

# 停止服务
openclaw gateway stop

# 重启服务（改了配置后要重启）
openclaw gateway restart

❓ 常见问题

花钱吗？

OpenClaw 免费开源。DeepSeek 的 API 非常便宜，日常使用每月大概 ¥5-20。

需要翻墙吗？

DeepSeek 不需要翻墙。但 Telegram 在国内需要，你可能需要准备一个科学上网工具。

安全吗？数据会泄露吗？

OpenClaw 跑在你自己电脑上，聊天记录存在本地。唯一发到外部的是你发给 AI 的对话内容（发到 DeepSeek 的服务器处理）。

电脑关机了 AI 还能用吗？

不能，OpenClaw 运行在你的电脑上，电脑需要开机且运行着。如果想 24 小时在线，可以跑在云服务器上（进阶操作）。
