📋 你需要准备的东西

  • 一台 Mac 或 Windows 电脑
  • 能上网（部分步骤可能需要科学上网）
  • 一个手机号（注册用）
  • 大约 30 分钟的时间

第一步 选一个 AI 模型并注册 ⏱ 5 分钟

下面三个都是国内能直接用的中文 AI 模型，选一个就行。不知道选哪个？直接选第一个。

🏆 选项 A：DeepSeek（推荐）

中文最强 · 最便宜 · 充 ¥10 用几个月

  1. 打开 platform.deepseek.com，手机号注册
  2. 点左侧 "API Keys" → "创建 API Key"
  3. ⚠️ 复制 Key 保存到备忘录（只显示一次！）
  4. 点左侧 "费用"，充值 ¥10

🅱️ 选项 B：智谱 GLM-4.7

清华团队出品 · 中文理解优秀 · 新用户有免费额度

  1. 打开 open.bigmodel.cn，手机号注册
  2. 进入控制台 → 点 "API Keys" → 创建 Key
  3. ⚠️ 复制 Key 保存到备忘录
  4. 新用户通常有免费额度，之后按需充值

🅲 选项 C：豆包 Doubao Pro 2.0

字节跳动出品 · 对话自然 · 价格便宜

  1. 打开 火山引擎 Ark 平台，注册登录
  2. 在"模型推理"中找到 Doubao-pro-32k，点"开通"
  3. 进入 "API Key 管理" → 创建 Key
  4. ⚠️ 复制 Key 保存到备忘录
  5. 新用户有免费额度，之后按需充值
⚠️ 重要：API Key 就像密码，不要发给别人，不要发到群里。

第二步 安装 Node.js ⏱ 5 分钟

这是 OpenClaw 运行需要的基础软件，装一次就行。

  1. 打开 nodejs.org/zh-cn
  2. 点绿色的 "下载 LTS" 按钮
  3. 双击下载的文件，一路点"继续/下一步"装完
✅ 验证：打开终端（Mac 按 ⌘+空格 搜"终端"；Windows 按 Win+R 输入 cmd），输入： 
node -v
看到类似 v22.x.x 就说明装好了。

第三步 安装 OpenClaw ⏱ 5 分钟

在刚才的终端窗口里，复制粘贴以下命令，按回车：

npm install -g openclaw

等它跑完（可能要 1-2 分钟），然后输入：

openclaw init

它会问你一些问题，按提示填就行。

第四步 配置 AI 模型 ⏱ 5 分钟

告诉 OpenClaw 用哪个 AI 模型。根据你第一步选的模型，找到对应的配置复制粘贴就行。

💡 先打开配置文件：
Mac：在终端输入 open ~/.openclaw/config.yaml
Windows：在终端输入 notepad %USERPROFILE%\.openclaw\config.yaml

把下面对应你选的模型的内容，整段复制粘贴到配置文件里（把 你的Key粘贴到这里 换成你第一步保存的 Key）：

🏆 选了 DeepSeek？复制这段：

model: deepseek/deepseek-chat

providers:
  deepseek:
    apiKey: "你的Key粘贴到这里"
    baseUrl: https://api.deepseek.com/v1

🅱️ 选了智谱 GLM-4.7？复制这段：

model: zhipu/glm-4-plus

providers:
  zhipu:
    apiKey: "你的Key粘贴到这里"
    baseUrl: https://open.bigmodel.cn/api/paas/v4

🅲 选了豆包 Doubao？复制这段：

model: doubao/doubao-pro-32k

providers:
  doubao:
    apiKey: "你的Key粘贴到这里"
    baseUrl: https://ark.cn-beijing.volces.com/api/v3

粘贴完保存文件就行，就这么简单。

第五步 创建 Telegram 机器人 ⏱ 5 分钟

通过 Telegram 跟你的 AI 助理聊天，随时随地，手机电脑都能用。

  1. 下载 Telegram（手机或电脑都行）
  2. 在 Telegram 搜索 @BotFather，给它发消息
  3. 发送 /newbot
  4. 它会让你起个名字和用户名，随便取
  5. 复制它给你的 Token（一长串字符）
  6. 再搜索 @userinfobot，发任意消息，它会回复你的数字 ID

把这些信息加到配置文件里：

channels:
  telegram:
    token: "你的Bot Token"
    allowedUsers:
      - 你的数字ID

第六步 启动！ ⏱ 1 分钟

一切配置好了，在终端输入：

openclaw gateway start

然后打开 Telegram，找到你创建的机器人，发一条消息试试！

🎨 让 AI 更懂你（可选）

OpenClaw 有几个特别的文件，可以让 AI 记住你的信息、按你喜欢的风格说话：

👤 USER.md — 你是谁

告诉 AI 你的基本信息，它就能更好地帮你。打开 ~/.openclaw/workspace/USER.md，写入：

- 名字：王帆
- 角色：产品经理
- 公司/行业：（你的行业）
- 关注领域：AI 产品化、用户体验
- 语言：中文

🤖 SOUL.md — AI 的性格

让 AI 用你喜欢的方式说话。打开 ~/.openclaw/workspace/SOUL.md，写入：

## 核心原则
- 全程用中文交流
- 说人话，别搞技术黑话
- 像靠谱的同事一样给建议
- 简洁为主，废话少说

## 风格
- 务实，不要假大空
- 有观点，别和稀泥
- 适当幽默，但别尬

🔧 几个常用命令

不用记，用到的时候回来看就行：

# 查看运行状态
openclaw status

# 启动服务
openclaw gateway start

# 停止服务
openclaw gateway stop

# 重启服务（改了配置后要重启）
openclaw gateway restart

❓ 常见问题

花钱吗？

OpenClaw 免费开源。DeepSeek 的 API 非常便宜，日常使用每月大概 ¥5-20。

需要翻墙吗？

DeepSeek 不需要翻墙。但 Telegram 在国内需要，你可能需要准备一个科学上网工具。

安全吗？数据会泄露吗？

OpenClaw 跑在你自己电脑上，聊天记录存在本地。唯一发到外部的是你发给 AI 的对话内容（发到 DeepSeek 的服务器处理）。

电脑关机了 AI 还能用吗？

不能，OpenClaw 运行在你的电脑上，电脑需要开机且运行着。如果想 24 小时在线，可以跑在云服务器上（进阶操作）。
